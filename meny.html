    Les specialités Africaine
    jour Plats Description
    Lundi Doro wat (Ethiopie) C'est une spécialité éthiopienne. Le doro wat, est
    une sorte de ragoût à base de poulet
    dans une sauce bien épicée.
    Mardi Le Djenkoumé (Togo) Voici un repas togolais délicieux et facile à manger.
    Il est fabriqué à partir de poulet, de semoule de blé,
    de tomates et d’épices.
    Mercredi Couscous Marocain (Maroc) Tc’est un plat très apprécié de la royauté.
    Jeudi Le Chakalaka (Afrique du Sud) C'est un plat traditionnel qui peut être comparé
    à la ratatouille. Il est composé de plusieurs
    types de légumes
    Vendredi Poisson attiéké (Côte d’Ivoire) Il est fabriqué à partir de manioc séché
    Samedi Le Foutou (Ghana) Il est fabriqué à partir de manioc
    de plantain et d’igname broyé dans un mortier.
    Dimanche Le Ndolè (Cameroun) Il est fabriqué à partir de légumes
